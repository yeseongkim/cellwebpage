# CELL Website (Static)

This repository contains the static website for CELL, prepared for direct deployment on Vercel.

## Deploy On Vercel

1. Push this repository to GitHub.
2. Import it in Vercel.
3. Use these settings:
   - Framework Preset: `Other`
   - Build Command: *(empty)*
   - Output Directory: `.`
4. Deploy.

## Notes

- URLs are relative and domain-agnostic.
- Main routes are organized by page folder (e.g. `/professor/`, `/members/`, `/research/`).
